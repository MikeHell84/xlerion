# Paleta de Colores - El Ropero Mag&co

## Colores Principales (Basado en PaletaColoresMag&co.png)

### Primarios

- **Negro**: `#000000` - Fondo principal y textos oscuros
- **Blanco**: `#FFFFFF` - Texto principal y fondos claros
- **Cyan/Azul Claro**: `#00e9fa` - Acentos, botones, highlights

### Secundarios

- **Gris Oscuro**: `#333436` - Fondos de cards, headers
- **Gris Medio**: `#4B5563` - Bordes y divisores
- **Gris Claro**: `#9CA3AF` - Texto secundario, metadatos
- **Gris Clara**: `#D1D5DB` - Texto terciario

### Colores de Estado

- **Verde**: `#10B981` - Activo, disponible, success
- **Rojo**: (reservado para errores)
- **Naranja**: (reservado para advertencias)

## Tipografía

- **Familia**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **Heading 1**: 48px, bold (hero titles)
- **Heading 2**: 32px, bold (section titles)
- **Heading 3**: 20px, bold (card titles)
- **Párrafo**: 16-18px, normal (body text)
- **Label**: 14px, normal (metadata)
- **Small**: 12px, normal (badges, status)

## Espaciado

- **Sección vertical**: 64px padding top/bottom
- **Contenedor ancho máximo**: 1200px
- **Padding horizontal**: 20px
- **Gap entre elementos**: 24px, 32px

## Bordes y Radio

- **Border Radius**: 8px (botones), 12px (cards)
- **Border Color**: `rgba(0, 233, 250, 0.2)` (subtle cyan)

## Sombras

- **Card Hover**: `0 0 30px rgba(0, 233, 250, 0.3)` (cyan glow)
- **Button Hover**: `0 0 20px rgba(0, 233, 250, 0.5)`
- **Status Badge**: `0 4px 20px rgba(0, 0, 0, 0.5)`

## Animaciones

- **Transiciones**: 0.3s ease
- **Pulse**: 2s infinite (status dot)

## Usado en

- `/public/index.html` - Frontend principal
- Logo: `LogoRopero.png`
- Favicon: `favicon.svg`, `favicon.png`, `favicon.ico`

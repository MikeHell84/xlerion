# Desarrollo de Videojuegos / Video Game Development

## ES — Resumen (máx 2 páginas)

**Alcance**: GDD, prototipado, programación, integración y publicación.  
**Fases**: Discovery → Diseño → Desarrollo → Integración → QA → Publish.  
**Duración típica**: 4–32 semanas según niveles y multiplayer.  

**Supuestos**

- Pilares de juego definidos.
- Herramientas y engine acordados.
- Acceso a plataformas de publicación.

**Riesgos**

- Scope creep de features.
- Rendimiento en dispositivos objetivo.
- Rechazo de stores.

**Recomendaciones**

- Sprint 0 obligatorio si no hay visión clara.
- Vertical slice temprano para validar diversión.
- QA reforzado para multiplayer.

## EN — Summary (max 2 pages)

**Scope**: GDD, prototyping, programming, integration and publishing.  
**Phases**: Discovery → Design → Development → Integration → QA → Publish.  
**Typical duration**: 4–32 weeks depending on levels and multiplayer.  

**Assumptions**

- Game pillars defined.
- Tools and engine agreed.
- Access to publishing platforms.

**Risks**

- Feature scope creep.
- Performance on target devices.
- Store rejections.

**Recommendations**

- Sprint 0 mandatory if vision is unclear.
- Early vertical slice to validate fun.
- Reinforced QA for multiplayer.

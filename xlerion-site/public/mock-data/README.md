# Mock Data para Demo de AI

Este directorio contiene imágenes de muestra para el demo de detección de plagas.

## Imágenes requeridas

1. `coffee-plant-sample.jpg` - Imagen principal de planta de café con plagas
2. `coffee-plant-sample-1.jpg` - Ejemplo con broca del café
3. `coffee-plant-sample-2.jpg` - Ejemplo con roya del café  
4. `coffee-plant-sample-3.jpg` - Ejemplo con múltiples plagas

## Instrucciones

Para que el demo funcione completamente, añade imágenes reales de plantas de café en este directorio.

Mientras tanto, el demo funcionará en modo sintético generando detecciones mock.

## Formato recomendado

- Formato: JPEG
- Tamaño: 800x600 a 1920x1080 px
- Peso: < 2MB por imagen
- Calidad: 80-90%

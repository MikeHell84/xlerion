Xlerion: Última Aurora (Preview)

Single-player narrative survival/action with adaptive soundtrack by Redemthor. Players explore ruins, engage in phase-based combat and make moral choices that change the world state.

Status: Early preview (2026)

Gameplay Specifications

Health
Base HP 100 (upgrades via implants)
Stamina
Sprint/evade resource: 100 units, regenerates over time
Skills
3 active skill slots (Shockwave, EMP burst, Ritual Slash)
Weapons
Melee (heavy/light), ballistic hybrid, ritual artifacts
Progression
Skill tree + weapon mod system + faction reputation
Objectives
Primary story missions + optional exploration contracts + dynamic world events
Save Model
Checkpoint + persistent world flags

Design Notes

  • Adaptive soundtrack driven by player state and region
  • Focus on environmental storytelling and emergent encounters
  • Balanced difficulty with options for stealth or direct combat
